# Homebrew Tap for Voice Prompt

This is the official Homebrew tap for [Voice Prompt](https://github.com/efeecllk/voice-prompt) - a lightweight macOS menu bar app for speech-to-text translation.

## Installation

```bash
brew tap efeecllk/voice-prompt
brew install --cask voice-prompt
```

## Requirements

- macOS 10.15+ (Catalina or later)
- Apple Silicon (M1/M2/M3)
- OpenAI API key ([Get one here](https://platform.openai.com/api-keys))

## Usage

After installation:
1. Click the Voice Prompt icon in your menu bar
2. Go to Settings and enter your OpenAI API key
3. Start recording!

## Uninstall

```bash
brew uninstall --cask voice-prompt
brew untap efeecllk/voice-prompt
```
